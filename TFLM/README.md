# wdcnn
文件夹内包含了STM32CubeMX工程文件和由其生成的MDK5工程。  
MDK工程使用的是STM32CubeMX自动生成的HAL库文件，根据注释修改和添加代码。
请注意，这是CubeAI里的TFLM部署方案，CubeAI只提供了C++到C的接口。
## 环境
### Windows
+ STM32CubeMX：V6.7.0
+ X-Cube-AI：V7.3.0
+ MDK：μVersion V5.38.0.0
+ ARM Compiler：V5.05
+ C Standard：C90
+ C++ Standard：C++ 11
+ ARM CMSIS：V5.9.0
+ ARM CMSIS-NN：V2.7.2
